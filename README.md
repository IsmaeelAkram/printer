# Printer
Printing functions for my Python projects.

## Installation & Usage
```sh
pip install git+https://github.com/ismaeelakram/printer
```
With emojis:
```python
>>> from printer import *
>>> good("Hey!")
```
![image](https://user-images.githubusercontent.com/19865145/126020113-532fd3d0-fb79-4272-97cc-a49e251fb454.png)

Without emojis:
```python
>>> from printer import *
>>> useEmojis(False)
>>> good("Hey!")
```
